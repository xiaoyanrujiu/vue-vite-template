# vue-vite-template 默认模板

# 目录说明

```
本项目
├─ README.md          # 说明文档
├─ public             # 静态资源
├─ src                # 源码
│  ├─ api             # 接口请求
│  ├─ assets          # vite打包的资源
│  ├─ components      # 公共组件
│  ├─ design          # ant-design ui组件注册
│  ├─ directive       # 自定义指令注册
│  ├─ enum            # 全局枚举
│  ├─ interface       # 全局接口类型定义
│  ├─ layout          # 全局Layout
│  ├─ router          # 路由
│  ├─ store           # 全局store管理
│  ├─ styles          # 全局样式
│  ├─ utils           # 全局公共方法
│  └─ views           # 所有业务页面
├─ tsconfig.json      # ts 编译配置
└─ vite.config.ts     # vite 配置
```
